Information

  • At the beginning, the server delays while generating the final network, this operation can be repeated if the resulting network is not connected
  • The first time you enter navigation may take some time to load
  • The game starts with 40000 starts with 1% probability of being inhabited with planets to share, sell or both, also there are 100 players distributed in 10 teams
  • Only the captain can finish the game
  • To change the probability and find more easily stars to trade with change the following backend fields:
    • DatabaseInit.java: line 80: prob = 2; line 90: nStars = 40;
    • GalaxyGraph.java: line 17: private final int MAX_EDGES = 40; line 18: private final int MAX_VERTICES = 40;
